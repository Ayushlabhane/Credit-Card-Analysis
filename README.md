# 💳 Credit Card Customer & Transaction Analysis

### 📖 Project Overview
This project focuses on analyzing credit card customer behavior and transaction patterns to understand revenue generation, spending habits, and demographic influences on credit card usage.

The analysis follows a structured data analytics workflow — starting from raw Excel datasets containing inconsistencies, performing data cleaning and transformation, analyzing the data using SQL, and finally visualizing insights through interactive Power BI dashboards.

The objective of this project is to convert raw financial and customer data into meaningful insights that help understand credit card revenue drivers and user spending behavior.

### 🎯 Purpose of the Analysis
The main goals of this analysis are to:
- Understand how different customer demographics contribute to credit card revenue
- Identify spending behavior across card categories and expenditure types
- Analyze how income, occupation, and age groups influence credit card usage
- Evaluate transaction trends and revenue distribution
- Present insights through clear and interactive dashboards

### 📊 Key Metrics Analyzed
- Total Credit Card Revenue
- Total Interest Earned
- Total Transaction Amount
- Total Transaction Count
- Revenue distribution by customer demographics
- Spending distribution by expenditure categories
- Revenue contribution by card category
- Revenue trends across time periods

### 🗂 Dataset Information
The dataset originally contained multiple Excel files with data inconsistencies and formatting issues. These raw files were cleaned and standardized before analysis.
The dataset includes information related to:
- Customer demographics (age, gender, income group, occupation)
- Credit card category
- Transaction amounts and counts
- Interest earned
- Expenditure types
- Geographic distribution of customers
- Cleaned datasets were converted into structured CSV files to perform analysis using SQL and to build Power BI dashboards.

### 🛠 Tools & Technologies Used
- Excel – Initial data inspection, cleaning, and formatting
- CSV – Structured data storage
SQL – Data querying and metric calculation
- Power BI – Interactive dashboards and visualization
- GitHub – Version control and project documentation

 
### 🔎 Business Questions Answered

1. Which credit card category generates the highest revenue?
2. What is the distribution of revenue across different income groups?
3. Which occupations contribute the most to credit card revenue?
4. Which age group generates the highest revenue?
5. Which states contribute the highest credit card revenue?
6. How does credit card usage vary between males and females?
7. Which expenditure categories account for the highest spending?
8. Which transaction method (Chip, Swipe, Online) generates the most revenue?

10. Which customer segments contribute the most interest income?

### 📌 Key Takeaways
- Certain customer occupations generate significantly higher credit card revenue
- Middle and high income groups contribute the majority of revenue
- Swipe transactions dominate credit card usage compared to chip and online payments
- Spending behavior varies significantly across expenditure categories
- Revenue distribution shows clear patterns across demographics and geography
